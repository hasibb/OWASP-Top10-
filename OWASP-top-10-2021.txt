# OWASP-Top10-
There are three new vulnerability added in 2021, which are Insecure Design, Software and Data Integrity Failures and Server Side Request Forgery [SSRF].

Here is the list for Top-10 vulnerabilities...
A01:2021-Broken Access Control
A02:2021-Cryptographic Failures
A03:2021-Injection
A04:2021-Insecure Design
A05:2021-Security Misconfiguration
A06:2021-Vulnerable and Outdate Components
A07:2021-Identification and Authentication Failures
A08:2021-Software and Data Integrity Failures
A09:2021-Security Logging and Monitoring Failures
A010:2021-Server-Side Request Forgery (SSRF)
